Migrating to Hexo

My old site was built on Angular, but with no backend it was a huge pain to update and manage. I went to update it and when I went to view my changes in the browser, the site wouldnt even render. Being tired of dealing with something so hard to manage I decided to look at Hexo. Hexo is a static site generator with built in deployment methods so I figured that would make my site much easier to manage. I was right but there were still some snags I hit that I couldnt find concrete answers to. So I figured I would add a post with my start to finish migration.

Requirements

Originally I tried to build everything from Ubuntu 16.04, despite the fact that every guide I could find recommended 14.04. I was consistently getting errors and unsupported architecture messages, so I reverted to 14.04. Amazingly, all of my problems vanished and everythin worked seamlessly. In other words, you will need Ubuntu 14.04.
